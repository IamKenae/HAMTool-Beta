## 📖 项目简介

HamTool Beta 是一款专为业余无线电爱好者（HAM）设计的现代化桌面应用程序的测试版本，集成了完整的通联日志管理、呼号查询、QTH地理位置管理等核心功能。基于 Electron + Vue 3 技术栈构建，提供跨平台支持和优雅的用户界面。软件支持多语言国际化、深色模式切换，为HAM们提供专业、高效、便捷的操作体验。

**注意：这是Beta版本，包含实验性功能，可能存在不稳定的情况。**

## ✨ 功能特性

### 🎯 核心功能 (Beta测试)
- **📝 通联日志管理** - 完整的QSO记录、编辑、搜索和统计分析
- **📚 呼号词典管理** - 内置呼号数据库，支持快速查询和批量管理
- **🌍 QTH地理位置** - 智能地理位置输入，支持拼音、中文和详细地区数据
- **💾 数据库系统** - 多数据库支持，数据备份恢复，版本冲突处理
- **🔄 数据导入功能** - 支持从正式版HAMTool导入数据到Beta版进行测试

### 🎨 用户体验
- **🌍 多语言支持** - 支持简体中文、英文，可扩展更多语言
- **🌙 深色模式** - 支持明暗主题自动切换和手动设置
- **📱 响应式设计** - 适配不同屏幕尺寸，支持移动端访问
- **⚡ 高性能** - 基于Electron + Vue 3构建，流畅运行
- **🔒 数据安全** - 本地数据存储，隐私保护，自动备份

### 🚀 实验性功能
- **📤 数据导入导出** - 支持JSON、Excel格式，带CC BY-NC-SA 4.0许可证保护
- **🔍 高级搜索过滤** - 多条件筛选、排序、全文搜索
- **📈 统计分析** - 通联数据统计、图表可视化
- **🌐 Web同步服务** - 实验室功能，支持局域网访问和数据同步
- **🏷️ 多种操作模式** - 点名模式和个人通联记录模式
- **🆕 新功能预览** - 提前体验正在开发的新功能

### 🛠️ Beta特性
- **🔧 独立更新渠道** - Beta版本独立更新，不影响正式版
- **📊 数据库隔离** - Beta版数据库与正式版完全独立
- **🌐 国际化框架** - 完整的i18n支持，翻译者信息显示
- **🔌 插件化架构** - 模块化设计，易于扩展

## 🚀 快速开始

### 安装注意事项

1. **备份正式版数据**: 在安装Beta版前，请备份正式版的重要数据
2. **独立安装**: Beta版可以与正式版同时安装，不会冲突
3. **数据隔离**: Beta版使用独立的数据库，不会影响正式版数据

### 首次使用

1. **许可证激活**: 首次启动时输入许可证密钥
   - 社区用户可以从激活网站获取免费许可证
   - 商业用户应联系支持获取商业许可证

2. **语言设置**: 选择首选界面语言
   - 进入 **设置** → **界面语言**
   - 从可用语言中选择（中文/英文）

3. **数据库设置**: 创建您的第一个QSO数据库
   - 进入 **数据库管理** → **新建数据库**
   - 为数据库输入描述性名称

### 从正式版导入数据

1. **正式版导出**: 在正式版HAMTool中导出JSON格式数据
2. **Beta版导入**: 在Beta版中使用 **从正式版导入** 功能
3. **选择数据**: 选择要导入的数据类型
4. **验证完整性**: 确认导入并验证数据完整性

## 💾 数据库管理

### 数据隔离说明

- **独立存储**: Beta版创建的数据库与正式版完全独立
- **版本隔离**: 不同Beta版本之间的数据库也是独立的
- **安全机制**: 导入操作不会影响源数据
- **测试友好**: 可以安全地测试新功能而不影响正式数据

### 数据导入流程

1. **导出准备**: 在正式版中导出需要测试的数据
2. **Beta导入**: 在Beta版中选择"从正式版导入"
3. **数据选择**: 选择要导入的数据库和记录类型
4. **确认导入**: 验证导入结果和数据完整性

### 重要提醒

- **单向导入**: 只支持从正式版导入到Beta版，不支持反向操作
- **数据备份**: 建议在测试前备份Beta版数据
- **版本兼容**: 不同Beta版本可能存在数据格式差异

## 🛠️ 技术架构

### 前端技术栈
- **框架**: Vue 3 (Composition API) + Element Plus
- **构建工具**: Vite + electron-vite
- **样式**: Bootstrap 5 + 自定义CSS + FontAwesome
- **国际化**: Vue I18n
- **状态管理**: Vue 3 响应式API

### 桌面应用
- **框架**: Electron (最新版本)
- **打包**: electron-builder
- **自动更新**: 独立的Beta更新渠道
- **跨平台**: Windows/macOS/Linux

### 数据存储
- **本地数据库**: IndexedDB
- **备份系统**: JSON/Excel导入导出
- **版本管理**: 数据库版本控制和迁移
- **数据隔离**: Beta版独立数据库系统
